# Digital Label Management System

A professional, full-stack application for managing digital labels.

## 📁 Project Structure

- **`frontend/`**: Next.js 16 application with Turbopack, Tailwind CSS, and Framer Motion.
- **`backend/`**: Utility scripts and administration tools. (Ready for API development).

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn

### Installation
From the root directory:
```bash
npm run install:all
```

### Development
Start the frontend development server:
```bash
npm run dev
```

### Administrative Scripts
Initialize Firebase collections:
```bash
npm run init-firebase
```

## 🛠️ Tech Stack
- **Frontend**: Next.js, TypeScript, Tailwind CSS, Radix UI, Framer Motion.
- **Backend/Scripts**: Node.js, Firebase Admin.
- **State Management**: Zustand.
