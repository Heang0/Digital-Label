1. Enable the Windows Subsystem features:
Copy and paste these two commands one by one and press Enter:

cmd
dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
cmd
dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
2. Download and Install Ubuntu 24.04:
After running the commands above, run this command to start the Ubuntu 24.04 download:

cmd
wsl --install -d Ubuntu-24.04
